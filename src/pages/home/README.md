# Главная страница (Home Page)

Главная страница системы, которая отображается пользователю после авторизации.

## Структура

```
home/
├── index.tsx                    # Точка входа
├── model/                       # Бизнес-логика
│   └── hooks/                   # React хуки для работы с данными
│       ├── useActiveTasks.ts    # Получение активных задач
│       ├── usePendingDocuments.ts # Получение документов на согласовании
│       ├── useUpcomingEvents.ts # Получение предстоящих событий
│       ├── useWelcomeStats.ts   # Получение статистики для приветствия
│       └── index.ts             # Экспорт хуков
└── ui/                          # UI компоненты
    ├── HomePage.tsx             # Главный компонент страницы
    ├── components/              # Переиспользуемые компоненты
    │   ├── ActiveTasks.tsx      # Карточка активных задач
    │   ├── PendingDocuments.tsx # Карточка документов на согласовании
    │   ├── UpcomingEvents.tsx   # Карточка предстоящих событий
    │   └── index.ts             # Экспорт компонентов
    └── sections/                # Секции страницы
        ├── AttentionCards.tsx   # Табы "Требует внимания"
        ├── QuickActions.tsx     # Панель быстрых действий
        ├── WelcomeSection.tsx   # Приветствие и статистика
        └── index.ts             # Экспорт секций
```

## Компоненты

### HomePage
Главный компонент страницы, объединяющий все секции.

### Секции (sections/)

#### WelcomeSection
- Приветствие пользователя (зависит от времени суток)
- Карточки со статистикой:
  - Активные задачи
  - Документы на согласовании
  - Всего документов
  - Завершенные задачи

#### AttentionCards
Табы с информацией, требующей внимания:
- **Документы** - документы на согласовании
- **Задачи** - активные задачи пользователя
- **События** - предстоящие события из расписания

#### QuickActions
Боковая панель с быстрыми действиями:
- Создать документ
- Создать задачу
- Открыть расписание

### Компоненты (components/)

#### PendingDocuments
Отображает список документов, ожидающих согласования:
- Название документа
- Отправитель
- Статус
- Дата создания
- Клик по карточке открывает детали документа

#### ActiveTasks
Отображает список активных задач:
- Название задачи
- Статус (Ожидает, В работе, На проверке)
- Дедлайн
- Участники задачи
- Клик по карточке открывает детали задачи

#### UpcomingEvents
Отображает предстоящие события из расписания:
- Название предмета
- Время
- Аудитория
- Преподаватель

## Хуки (model/hooks/)

### useWelcomeStats
Получает статистику для отображения в приветственной секции:
- Количество активных задач
- Количество документов на согласовании
- Общее количество документов
- Количество завершенных задач

**API:** `GET /tasks/employee`

### usePendingDocuments
Получает список документов, требующих согласования.

**API:** `documentsApi.getApplicationDocuments({ scope: 'inbox', limit: 10 })`

### useActiveTasks
Получает список активных задач пользователя (в работе, на проверке, ожидающие).

**API:** `GET /tasks/employee`

### useUpcomingEvents
Получает предстоящие события из расписания (сегодня или следующий день).

**API:** `GET /schedule`

## Использование

```tsx
import { HomePage } from 'pages/home';

// В роутере
<Route path="/home" element={<HomePage />} />
```

## Стилистика

Страница использует компоненты из shadcn/ui:
- Card, CardContent, CardHeader, CardTitle
- Button
- Badge
- Tabs, TabsList, TabsTrigger, TabsContent
- Avatar, AvatarImage, AvatarFallback
- Skeleton (для загрузки)

Цветовая схема:
- Синий - задачи
- Желтый - документы на согласовании
- Фиолетовый - события
- Зеленый - завершенные задачи

## Навигация

Клики по карточкам ведут на соответствующие страницы:
- Документы → `/documents/applications/:id`
- Задачи → `/task` (с передачей taskId в state)
- Быстрые действия используют `useFormNavigation` для открытия диалогов
